enchant: Adds enchantments to the item the player is currentlyholding. Specify 0 for the level to remove an enchantment. Specify force to ignore normal enchantment restrictions

gamerule: Sets a server's game rules
give: Gives the specified player a certain amount of items
help: help <page>						Shows the help menu
kick: kick <player_name>				Removes the specified player from the server
kill: kill <player_name>				Commits suicide, only usable as a player
list: list								Lists all online players
me: me <message>						Performs the specified action in chat
